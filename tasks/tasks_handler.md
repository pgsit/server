## Tasks Handler

## Requirements
-R1 Add tasks handler
-R2 Handler should send back all tasks from the todo database
-R3 Add weekly tasks handler. It should return tasks between now and a week
-R4 Add today tasks handler. It should return tasks for today
-R5 Add tomorrow tasks handler. It should return tasks for tomorrow
-R6 Do not return created_at and id info to the client
-R7 Show past tasks in Green color. Render name, description etc. in Green.
-R8 Tasks/View should have four center-aligned buttons with labels - All, Today, Tomorrow and Week

## Enhancements
-E1 Map /tasks to tasks/view
-E2 Show delete button next to each row and clicking it should delete that task and refresh the page
-E3 Do not show Delete option for tasks which are already past due date
