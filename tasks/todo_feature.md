## ToDo List Feature

## Requirements
-R1 Create todo database with date, time, description and username fields
-R2 Implement ToDo Handler
-R3 Handler presents multiline edit box and Calendar widget with time option
-R4 Also adds Name edit box
-R5 Data submitted in these widgets should be saved in todo database
-R6 Time should default to current time

## Validations
-V1 Do not allow submissions if any of the fields is empty
